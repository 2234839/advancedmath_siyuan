# 思源《高等数学》样板项目
下载页： https://github.com/tritium333/advancedmath_siyuan/releases 

现已更新至第一章第一节。

本项目具有如下特色：
1. 知识点之间的引用。比如书中凡是出现“连续”的地方，我都链接到“连续”这一定义所在块，方便温故知新。
2. 每个定理、定义都制成闪卡。
3. 每个定理、定义都链接到 pdf 教材的原文位置。

可随意用于教学、学习、科研、思源的宣传推广、教程制作等用途。
